Import crash logs, symbolicate quickly, and find root causes.

A Crash Analysis Tool for iOS/macOS Developers

Crash Explorer streamlines your workflow from log import and dSYM matching to symbolication and root-cause triage.

Core Features

  • Import and parse .ips / .crash files
  • Automatic dSYM discovery, UUID matching, and manual selection
  • Visualized threads, stack frames, and binary images
  • Manual address symbolication (offset supported)
  • Optional AI insights (user-configured OpenAI-compatible service)

Use Cases

  • Daily client-side crash triage
  • QA crash attribution and regression verification
  • Fast diagnosis for recurring production crashes
