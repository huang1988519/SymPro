Crash Explorer Support

Last updated: 2026-03-24

Thanks for using Crash Explorer. If you encounter any issue, contact us via:

Support email: support@anti.xin

When reporting an issue, include

  • App version (from Settings)
  • macOS version and device model
  • Issue description and reproduction steps
  • Relevant logs/screenshots (redacted if needed)

FAQ

Q: AI insight unavailable?
A: Verify endpoint/model/API key in Settings and ensure the service is reachable.

Q: Imported .crash shows no content?
A: Ensure the file is a valid crash report with thread stack and Binary Images sections.

Q: dSYM does not match?
A: Check whether dSYM UUID is consistent with crash image UUID.
