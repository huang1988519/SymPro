Privacy Policy

Last updated: 2026-03-24

This Privacy Policy explains how SymPro handles data when you use the app.

1. Data Processing Scope

SymPro is a developer tool for local crash analysis. Crash logs and dSYM files are processed locally on your device by default.

2. AI Feature (Optional)

If you enable AI insight and configure a third-party OpenAI-compatible service, selected crash report content will be sent to that service for analysis. This behavior is user-initiated and optional.

3. API Keys

API keys entered by users are stored in the system Keychain and used only for user-initiated AI requests.

4. Data Sharing

SymPro does not sell personal data. Data is only transmitted to third-party AI providers that users configure themselves.

5. Contact

For privacy-related questions, contact: support@anti.xin
